Welcome to WAF game

Description:
You could upload a file written by PHP.
And then I will use the file you uploaded to check some other files.
These files include PHP webshell files, normal PHP files, PHP files(syntax error).
Here are some demos 

<?php
//not webshell
echo 123;
?>
<?php 
//webshell
eval($_GET[1]);
?>
<?php
//not webshell (syntax error)
eval($_GET[1])
Now, you need to distinguish them.
You can receive these files through "file_get_contents('php://input')", I will use your script to check them one by one.
If it is a PHP webshell file, you should output "webshell", else you should output "no-webshell".
Here is a sample WAF file. 
<?php
$case = file_get_contents('php://input');
if(strpos($case,'eval')!==false){
    die('webshell');
}
else{
    die('no-webshell');
}
?>
